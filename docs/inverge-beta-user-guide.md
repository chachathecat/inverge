# Inverge 클로즈드 베타 사용자 가이드

초대받은 감정평가사 수험생(1차/2차)을 위한 짧은 사용 안내입니다.

## Inverge는 무엇인가요?
Inverge는 감정평가사 시험 준비를 위한 **학습 운영 시스템**입니다.
입력한 학습 기록을 바탕으로, 다음 행동(복습/재작성/재시도)을 더 분명하게 안내합니다.

## Inverge는 무엇이 아닌가요?
- 최종 합격을 자동 판정하는 서비스가 아닙니다.
- 사람 검토 없이 결과를 확정하는 자동 채점기가 아닙니다.
- 단순 점수 확인으로 끝나는 도구가 아닙니다.

## 1차 시작 방법
1. 로그인 후 1차 모드(감정평가사 1차)를 선택합니다.
2. 오늘 할 일에서 1차 학습 시작 버튼으로 진입합니다.
3. 세트 풀이 또는 빠른 오답 기록 중 현재 상황에 맞는 흐름을 선택합니다.

## 1차 세트 풀이 사용법
1. 세트 문제를 풀고 답안을 입력합니다.
2. 오답 항목에서 오답 이유와 회상 내용을 간단히 남깁니다.
3. 완료 후 review queue에서 다시 볼 항목을 확인합니다.

## 1차 빠른 오답 기록 사용법
- 문제를 길게 풀기 어려운 상황에서는 빠른 오답 기록을 사용하세요.
- 핵심은 “틀린 이유 1개 + 바로 할 다음 행동 1개”를 남기는 것입니다.
- 기록 후 review queue에서 짧게 재시도하면 학습 손실을 줄일 수 있습니다.

## 2차 시작 방법
1. 로그인 후 2차 모드(감정평가사 2차)를 선택합니다.
2. 2차 작성 워크스페이스로 진입합니다.
3. 쟁점 회상 → 목차 → 답안 작성 → 비교/간극 확인 → rewrite 순서로 진행합니다.

## 2차 writing workspace 사용법
- 한 번에 완벽하게 쓰기보다, 단계별로 짧게 작성하고 바로 수정합니다.
- 간극(내 답안 vs 기준 관점)에서 가장 큰 1개를 먼저 잡고 rewrite 하세요.
- 저장 실패가 보이면 새로 진행하지 말고 먼저 저장 상태를 확인한 뒤 재시도하세요.

## Review queue 해석 방법
- review queue는 “틀렸던 문제 목록”이 아니라 **다음 학습 행동 목록**입니다.
- 우선순위가 높은 항목부터 짧게 재시도하고, 필요 시 다시 rewrite 하세요.
- 점수보다, 다음 행동이 분명한 상태를 목표로 사용하면 효과가 큽니다.

## 중요한 안내
- Inverge 결과는 학습 보조 정보이며, **최종 채점/최종 판정 권한이 아닙니다.**
- 화면이 헷갈리거나 저장이 실패하면, 해당 화면과 상황을 베타 운영팀에 바로 알려주세요.
