# Five-Year Reference QA (2021-2025)

## 목적
- 2021~2025 감정평가사 2차 기출 레퍼런스 레이어가 알파 사용자 기준에서 **유용하고 보수적이며 안정적인지** 수동 점검한다.
- 본 문서는 QA 체크리스트 전용이며, 신규 연도(2020/2019) 추가를 포함하지 않는다.

## 사전 조건
- 레퍼런스 seed는 2021/2022/2023/2024/2025만 포함한다.
- 모든 레퍼런스는 다음 정책을 유지한다.
  - `source_status: "needs_review"`
  - `raw_text_policy: "reference_only"`
- 점검 범위는 학습자 표면(캡처 → 아이템 상세 연결)만 대상으로 한다.

## 수동 QA 체크리스트

### 과목별 연결 기본 확인
- [ ] 실무 메모를 남기면 관련 레퍼런스가 노출된다.
- [ ] 이론 메모를 남기면 관련 레퍼런스가 노출된다.
- [ ] 법규 메모를 남기면 관련 레퍼런스가 노출된다.

### 연결 설명 품질 확인
- [ ] `연결 이유` 문구가 학습자용 한국어로 이해 가능하다.
- [ ] `연결된 신호`가 과목/논점/오류/구조 맥락과 일치한다.
- [ ] 원점수(raw score)나 채점 수치가 UI에 노출되지 않는다.

### skeleton 가이드 품질 확인
- [ ] `학습용 skeleton 단계`가 표시된다.
- [ ] `자가 점검 질문`이 표시된다.
- [ ] `자주 발생하는 간극`이 표시된다.
- [ ] `다음 행동`이 표시된다.

### 안전 가드레일 확인
- [ ] 전체 기출 아카이브(full archive) UI가 노출되지 않는다.
- [ ] 공식 모범답안/공식 채점/합격-불합격(pass-fail) 언어가 노출되지 않는다.
