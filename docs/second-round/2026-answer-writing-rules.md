# 2026 논술형 답안지 작성 규칙 정리

## 답안지 구성

- 표지
- 연습지
- 답안내지 20쪽
- 답안지는 분리 또는 훼손 금지

## 필기구 규칙

- 지워지지 않는 검은색 필기구만 사용
- 지워지는 펜, 연필류, 유색 필기류, 2색 이상 혼합 사용 시 해당 답항 0점 처리 위험

## 인적사항/부정표시 규칙

- 수험번호 및 성명은 연습지 첫 장 좌측 인적사항란에만 작성
- 인적사항란 외 성명·수험번호·특수표시·특정인 암시 표시 금지
- 위반 시 답안지 전체 무효 또는 0점 처리 위험

## 계산문제 작성 규칙

- 계산과정 기재
- 답 기재
- 단위 기재
- 실무형 채점 엔진에서 `계산과정/답/단위 누락`은 별도 형식 리스크로 표시

## 답안 구성 규칙

- 문제번호 및 문제를 기재한 뒤 답안 작성
- 문제번호 순서와 관계없이 작성 가능하나, 어떤 문제 답안인지 명확히 표시
- 요구한 가지 수 이상을 쓰면 답란 기재 순서대로 요구 수까지만 채점
- 각 문제 답안 끝: `끝`
- 최종 답안 끝: `이하여백`
- 추가 답안지 사용 시 기존 답안지 20쪽 우측하단 `[ ]`에 `계속` 표시

## 공개 여부

- 채점기준 및 모범답안은 공개하지 않는다고 명시되어 있음
- 따라서 2025년 제36회 2차 자료의 `modelAnswer`는 공식 모범답안으로 표시하면 안 됨
