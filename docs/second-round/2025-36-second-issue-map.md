# 2025년 제36회 감정평가사 2차 논점맵 / Skeleton 초안

주의: 사용자 제공 PDF 기반 문제 원문 정제본이다. 공식 모범답안 또는 공식 채점기준이 아니다.
모든 Skeleton은 문장형 모범답안이 아니라 채점관 모드 입력용 목차·키워드 구조다.

## 감정평가실무

### 문제 1 / 40점 / practice

**핵심 논점 키워드**
- 공사중단 부동산
- 현상 평가
- 개발완료 가치
- 공시지가기준법
- 공사비
- 기성고
- 수익가격
- PI
- IRR
- NPV

**Skeleton Framework**
- Ⅰ. 현상 기준 매수가격: 표준지/거래사례/평가선례 비교가능성 검토, 표준지 기준 산정, 그 밖의 요인 보정, 토지단가 반올림
- Ⅱ. 공사중단 건물: 기성공사비, 공정률, 공사중단 감가 $기성건축공사비 \times 5\%$
- Ⅲ. 개발완료 매수가격: 추가공사안 vs 철거 후 신축안, 미성공사비 20% 증가, 신축 총공사비 25% 증가, 철거비 $기성건축공사비 \times 10\%$
- Ⅳ. 수익가격: 임대사례 선정, 임대료지수 보정, 지역/개별요인 비교, $NOI=유효총수익-영업경비$, $V=NOI/R$
- Ⅴ. 시장동향/투자지표: $PI=\frac{PV(현금유입)}{PV(현금유출)}$, $NPV=\sum\frac{CF_t}{(1+r)^t}-I_0$, $IRR: NPV=0$

**채점 위험 포인트**
- 산식 또는 도출과정 누락
- 단위/반올림/절사 조건 누락
- 자료 선택 근거 부재
- 초기 계산 오류의 파급효과와 별도 논점 오류 구분 필요

### 문제 2 / 30점 / practice

**핵심 논점 키워드**
- 개인기업 법인전환
- 영업관련 기업가치
- 영업권
- DCF
- FCFF
- WACC
- CAPM
- 운전자본
- 영업투하자본

**Skeleton Framework**
- Ⅰ. 매출 추정: 과거 3년 상승률, 고속성장기 5년, 안정성장기 $g=0$
- Ⅱ. 영업현금흐름: 매출원가율, 판매관리비율, 적정 임차료, 대표자 자가노력비, 소득세/법인세 조건 반영
- Ⅲ. 추가운전자본: $운전자본소요율=\frac{1}{매출채권회전율}+\frac{1}{재고자산회전율}-\frac{1}{매입채무회전율}$
- Ⅳ. 할인율: $r_e=R_f+eta(E(R_m)-R_f)+RP$, $WACC=\frac{E}{D+E}r_e+\frac{D}{D+E}r_d(1-T)$
- Ⅴ. 영업권: $영업권=영업관련\ 기업가치-영업투하자본$

**채점 위험 포인트**
- 산식 또는 도출과정 누락
- 단위/반올림/절사 조건 누락
- 자료 선택 근거 부재

## 감정평가이론

### 문제 1 / 40점 / theory

**핵심 논점 키워드**
- 일단지
- 담보평가
- 최유효이용
- 경제적 타당성
- 거래사례 검토
- 나무/건축허가 권리 포함

**Skeleton Framework**
- Ⅰ. 일단지: 인접 2필지, 동일 용도지역, 총 500㎡, 단독주택 건축허가, 명의변경신고필증, 표준적 획지 500㎡
- Ⅱ. 일단지 근거: 용도상 불가분성, 거래·이용·처분 일체성, 사회·경제·행정적 합리성
- Ⅲ. 최유효이용 경제적 타당성: 수요, 개발비, 수익, 위험, 금융조달, 순현재가치, 법적/물리적 가능성 통과 후 경제성
- Ⅳ. 거래사례 검토: 정상성, 사정보정, 시점수정, 지역요인, 개별요인, 나무 및 건축허가권 포함 여부, 기준시점 차이

**채점 위험 포인트**
- 개념 정의만 쓰고 사례 포섭 누락
- 문제의 세부 물음별 배점 안배 실패
- 기준/요건과 사안 사실의 연결 부족
