# Answer Review Field Trial Scorecard

## Purpose
Evaluate whether Answer Review results are useful enough for closed beta and paid pilot.

## Scoring scale
- 1 = unusable
- 2 = weak
- 3 = acceptable
- 4 = useful
- 5 = would pay / would reuse

## Evaluation dimensions
1. Korean clarity
   - Result is natural Korean.
   - No English leakage in main result.

2. Primary fix quality
   - “가장 먼저 고칠 1가지” is specific.
   - It matches the uploaded answer/problem.

3. Skeleton usefulness
   - Skeleton has useful 목차 and 필수 키워드.
   - It is not a full sentence model answer.
   - It is not generic.

4. Next action clarity
   - Next action can be done within 10 minutes.
   - User knows exactly what to rewrite or review.

5. Save motivation
   - User wants to save this result.
   - User understands it will feed 약점 신호, 복습 큐, 오늘 계획.

6. Trust/safety
   - No official grading/pass/fail impression.
   - Result clearly feels like a learning-aid draft.

## Overall paid-pilot threshold
- Average score >= 4.0 across at least 5 real samples.
- No critical failure in Korean clarity or trust/safety.
- At least 2 testers say they would use it again tomorrow.
- At least 1 tester says they would pay for it.

## Critical failures
- Main result is English.
- Result is unrelated to uploaded answer.
- Skeleton is empty or nonsense.
- Result implies official grading.
- User does not know what to do next.
