# Reference Matching QA (2023/2024/2025)

## 목적: reference matching이 하는 일
- 캡처 노트 신호(과목/논점 후보/오류 유형/구조 약점)를 기반으로 **학습용 관련 기출 후보**를 최대 3개까지 제시합니다.
- 각 후보에 대해 학습자가 바로 실행할 수 있는 **연결 이유 / 연결된 신호 / 학습용 skeleton / 자가 점검 질문 / 자주 발생하는 간극 / 다음 행동**을 제공합니다.
- 목적은 정답 제시가 아니라, 다시쓰기·재시도에 필요한 구조적 힌트를 빠르게 연결하는 것입니다.

## 비목적: reference matching이 하지 않는 일
- 공식 모범답안 제공
- 공식 채점/확정 판정
- 합격/불합격 판단
- 전체 기출 아카이브 탐색 UI 제공

## 수동 QA 체크리스트

### 1) 실무 신호 입력
- 캡처 노트 신호 예시:
  - 과목: 감정평가실무
  - 논점 후보: 평가방법 / 자료검토 / 시점수정
  - 오류 유형: 계산 실수 / 조건 누락
  - 구조 약점: 결론 / 단위 / 근거

### 2) 이론 신호 입력
- 캡처 노트 신호 예시:
  - 과목: 감정평가이론
  - 논점 후보: 개념정의 / 논거연결 / 사례적용
  - 오류 유형: 개념 혼동 / 구조 약함
  - 구조 약점: 결론 / 사례 적용

### 3) 법규 신호 입력
- 캡처 노트 신호 예시:
  - 과목: 감정평가 및 보상법규
  - 논점 후보: 요건 / 절차 / 사안포섭
  - 오류 유형: 암기 누락 / 조건 누락
  - 구조 약점: 조문 / 포섭 / 결론

### 4) 결과 확인
- 관련 기출 후보가 해당 과목 기준으로 상단에 노출되는지 확인
- 연결 이유 문구가 학습자 친화적 한국어인지 확인
- 연결된 신호(matched fields)가 비어 있지 않은지 확인
- 학습용 skeleton 가이드(단계/점검 질문/간극/다음 행동)가 모두 채워져 있는지 확인
- 공식 답안/공식 채점/합격·불합격 표현이 없는지 확인
- 전체 기출 아카이브를 탐색하는 UI가 생기지 않았는지 확인
