# Closed Beta Operator Runbook

1. Invite 3–5 users.
2. Ask each user to test:
   - public landing
   - anonymous answer review trial
   - signup/login
   - answer review after login
   - capture with photo
   - review queue
   - today plan
3. Collect:
   - first confusion point
   - whether result was useful
   - whether they would use again tomorrow
   - whether they would pay
4. Stop conditions:
   - OCR fails for most users
   - result quality is confusing
   - users cannot find next action
   - mobile input is broken
   - login/signup blocks trial conversion
