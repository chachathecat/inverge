# Source Text Pilot QA

## Purpose
source-text pilot verifies source → extraction candidate → structured candidate → reference linkage.

## Non-purpose
- not OCR
- not official text
- not official answer
- not official scoring
- not learner-facing archive

## Manual QA checklist
- [ ] pilot source id exists
- [ ] extraction candidate exists
- [ ] structured candidate exists
- [ ] source id resolves
- [ ] reference id resolves
- [ ] both candidates are `reference_only`
- [ ] both candidates are `needs_review`
- [ ] no reviewed status
- [ ] no OCR/upload/archive UI
